## Overview

MacWhisper Professional Suite is a complete local transcription platform with batch processing, export flexibility, and editing tools for professional transcription workflows.

Batch processing transcribes multiple audio and video files automatically with consistent settings, handling the volume processing that professional transcription workflows generate. The batch queue processes files sequentially or in parallel depending on hardware capability.

Export formats produce transcripts as plain text, SRT subtitles, VTT captions, CSV with timestamps, and other formats suitable for different downstream uses. The export options handle the format requirements of video editing, accessibility compliance, and document archival.

Transcript editing provides an integrated text editor with audio playback synchronized to the transcript, enabling correction and refinement while listening to the original audio. Click-to-navigate plays the audio from any point in the transcript.

MacWhisper Professional Suite is the transcription platform for professionals who need accurate, private, local transcription at production volume with the export flexibility and editing tools that professional transcription work demands.

## Performance & Integration

MacWhisper is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while MacWhisper processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
